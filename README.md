## 📋 项目简介

FindTeacherIP 是一个用于研究和分析极域电子教室网络通信协议的工具。它可以自动发现教师机IP地址，并提供多种协议测试模式来演示网络通信的安全性。

**⚠️ 重要提醒：**
- 本项目仅用于网络安全教育和研究目的
- 请在授权环境下进行测试
- 非法使用可能违反相关法律法规
- 项目大部分代码由AI生成，仅供参考学习

## 🛠️ 功能特性

### 🔍 教师机发现
- **自动扫描**：监听组播地址 `224.50.50.42:4988`
- **多协议支持**：识别 OONC 和 CANC 数据包
- **智能解析**：从网络流量中提取教师机IP地址
- **实时显示**：显示发现过程和网络状态

### ⚡ 攻击演示模式

#### 1. FILESUBMIT协议崩溃攻击
- **协议分析**：基于抓包数据的576字节FILESUBMIT畸形包
- **目标端口**：教师机控制端口 `4806`
- **技术特点**：模拟协议级异常数据处理
- **教育价值**：演示协议安全性的重要性

#### 2. 不客气模式（持续骚扰）
- **模拟行为**：交替发送举手包和随机消息包
- **目标端口**：教师机消息端口 `5512`
- **通信模拟**：基于实际抓包数据的协议结构
- **研究用途**：分析系统在高负载下的表现

## 🔧 技术细节

### 网络协议
- **组播通信**：`224.50.50.42:4988`
- **控制协议**：TCP `4806` 端口
- **数据协议**：UDP `5512` 端口

### 数据包结构
基于实际网络抓包分析：
- OONC心跳包（44字节）
- CANC控制包（112字节）
- FILESUBMIT文件提交包（576字节）
- TEXTSUBMIT文本提交包（971字节）

### 颜色编码输出
- 🔵 **蓝色**：信息和状态消息
- 🟢 **绿色**：成功和确认消息  
- 🟡 **黄色**：警告和注意消息
- 🔴 **红色**：错误和失败消息
- 🟣 **紫色**：标题和重要信息
